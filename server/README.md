# Node-Crud
Node js(ES6) CRUD operation using mongoose

# Download and clone the repository 

git clone https://github.com/senthilmca90/Node-Crud.git

go to inside the folder
>cd Node-Crud
# install packages 
>npm install

Connect your mongodb

# start the node server
> node server

# check this url
http://localhost:3000
